#### 问题 1: px4_msgs 构建失败

**状态**: 🔴 BLOCKED
**严重程度**: ⚠️ 中等 (可降级绕过)
**影响范围**: PX4 Offboard 消息发布
**原因**: Rust 模板生成器错误 (em.py TransientParseError)
**当前解决方案**: 实现了可选依赖降级模式
```python
try:
    from px4_msgs.msg import ...
    PX4_MSGS_AVAILABLE = True
except:
    PX4_MSGS_AVAILABLE = False
    # 继续安全运行
```

**解决计划**:
1. 调查 Rust 模板生成器配置
2. 或禁用 Rust 生成器，仅生成 Python
3. 或使用预构建的 px4_msgs 二进制包

**优先级**: `HIGH` (需在硬件验收前解决)
